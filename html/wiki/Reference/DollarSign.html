$ (cast/if-else/dot product)

cast

The $ (cast) dyadic operator converts strings or other data to another type of data. For example, we might use it to parse a string as an integer. The left hand argument is the type to which to convert the right hand argument.

A common mistake is to use a small letter type specifier as the left hand argument when converting from a string. You must use a capital type letter when converting from strings.

The cast operator is overloaded and so has other meanings. One is to right or left pad a string with spaces. In this case the left hand argument is a positive integer to add spaces to the right of the string or a negative value to pad on the left.

Example

Converting a string to a number: 

q)"I" $ "3"
3
q)"F" $ "3.14"
3.14

Converting from a string to a string: 

q)"c" $ "foo"
"foo"

Converting from string to symbol and back: 

q)`$"ibm"
`ibm
q)string `ibm
"ibm"

Converting characters to ASCII codes and back: 

q)"i"$"abcde"
97 98 99 100 101
q)"c"$97 98 99 100 101
"abcde"

Converting from one type of number to another: 

q)"i" $ 3.14
3
q)"f" $ 3
3f

Note we can also use the type names or type numbers as well as the type specifiers: 

q)`int$3.14
3
q)1h$1 0 2
101b

The later is useful if you want to cast to the type of an existing variable 

q)x:1 2 3j
q)(type x)$10 20 30
10 20 30j

We can use the $ operator to pad a string as follows, recalling that negative means pad on the left while positive means pad on the right. 

q)10$"foo"
"foo       "
q)-10$"foo"
"       foo"

Partial applications of the $ operator can of course be constructed. Here we make a function f that adds 10 spaces to the right of the given string. 

q)f:10$
q)f "foo"
"foo       "
q)f "bar"
"bar       "

datetime example 

`datetime`year`month`week`mm`dd`hh`uu`ss$.z.Z
"D"$"yyyy/mm/dd"
in general:
(where / can be /-. or blank)
[yy]yymmdd
ddMMM[yy]yy
yyyy/[mm|MMM]/dd
[mm|MMM]/dd/[yy]yy  / \z 0
dd/[mm|MMM]/[yy]yy  / \z 1
MMM is jan..dec

minute,second and time string formats 

minute "U"$"hh:mm"
second "V"$"hh:mm:ss"
time "T"$"hh:mm:ss.mmm"

"U"$ "V"$ "T"$ also allow the above but without the ':'s

"T"$ also allows hhmmssmmm and hh:mm:ss.mmm
"V"$ also allows hh:mm(promote) and hh:mm:ss.mmm(trunc)

timestamp from milliseconds (e.g. java:currentTimeMillis) 

q)/scale up to nanos, change epoch, and cast back
q){"p"$1970.01.01D + 1000000j * x}1317614494097j
2011.10.03D04:01:34.097000000

matrix product (mmu)

calculate the product of two float matrices 

q)(1 2 3f;4 5 6f)$(7 8f;9 10f;11 12f)
58  64
139 154

this is the same as the mmu function.
