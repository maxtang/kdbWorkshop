type (type function)

The function type returns the type of its argument as a short integer. Negative numbers are for atoms, positive numbers are for lists, and zero is a general K list. The type codes are as follows:

0h list
1 boolean
2 guid (since v3.0)
4 byte
5 short
6 int
7 long
8 real
9 float
10 char
11 symbol
12 timestamp
13 month
14 date
15 datetime (deprecated, use timestamp)
16 timespan
17 minute
18 second
19 time
20-76 enums
78-96 77+t - mapped list of lists of type t
98 table
99 dictionary
100 lambda
101 unary primitive
102 binary primitive
103 ternary(operator)
104 projection
105 composition
106 f'
107 f/
108 f\
109 f':
110 f/:
111 f\:
112 dynamic load

For a complete list of functions, see the kdb+ Function Reference.
